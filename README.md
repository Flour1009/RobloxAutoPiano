# RobloxAutoPiano
Скрипт читает MIDI-файл и превращает ноты в нажатия клавиш клавиатуры. Он может играть только мелодию или несколько нот сразу. Подходит для автопроигрывания виртуального пианино, например в Roblox. Запуск и остановка выполняются горячими клавишами. Также сам подбирает диапазон и транспонирование нот.
